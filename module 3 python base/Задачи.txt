1️⃣ Функции — Склад товаров

Напиши функцию update_inventory, которая:

Условия:

Принимает:
current_inventory — словарь {название товара: количество}
shipment — словарь {название товара: количество}
Добавляет количество из поставки к текущему складу.
Если итоговое количество товара > 1000, установи максимум = 1000.
Функция возвращает новый словарь склада.

Пример:

current_inventory = {"яблоки": 500, "бананы": 200}
shipment = {"яблоки": 600, "бананы": 100, "груши": 50}

print(update_inventory(current_inventory, shipment))
# {'яблоки': 1000, 'бананы': 300, 'груши': 50}
2️⃣ Класс и методы — Банк

Создай класс BankAccount с:

Атрибутами:
account_number
balance
Методами:
deposit(amount) — добавить деньги (только если >0)
withdraw(amount) — снять деньги (если хватает баланса)
transfer_to(other_account, amount) — перевести деньги на другой счёт (с проверкой баланса)
В случае ошибок (недостаточно средств или отрицательная сумма) — выводить понятное сообщение.

Дополнительно:
Добавь метод apply_interest(rate), который увеличивает баланс на balance * rate.

4️⃣ Полиморфизм — Отчёты

Создай два класса отчётов:

SalesReport → метод generate() возвращает "Генерация отчёта по продажам"
InventoryReport → метод generate() возвращает "Генерация отчёта по складу"

Функция create_report(report) принимает объект отчёта и вызывает метод generate().
Пример работы:

create_report(SalesReport())      # Генерация отчёта по продажам
create_report(InventoryReport())  # Генерация отчёта по складу
